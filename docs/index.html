Sensing & IoT

Project Objectives

This web app is for my fourth year Sensing and Internet of Things coursework, where I collected sensor data from a user-centred environment, my toilet, as well as live cryptocurrency prices from the CoinGecko API. The data was collected over a period of 7 days, processed in Juptyer Notebook using the pandas library. To find the code used in this project please refer to my GitHub.
